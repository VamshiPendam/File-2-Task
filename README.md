This project allows users to upload a file, automatically reads its content, and breaks it down into smaller, manageable tasks for easier understanding and regular study. It features an integrated AI assistant that helps users with queries related to the content. The platform also includes a progress analysis system to track and evaluate task completion effectively.
=> Technologies Used:
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB
Additional Features: AI assistant for query support and task analysis system
